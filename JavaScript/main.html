JavaScript Basics: Arrays

Introduction to Arrays

  • var array = [ item1, item2, item3];
  • Add items: 
    array[array.length] = newItem;
    or 
    brarray.push(newItem)
  • Delete an item in the end: 
    array.pop();
  • Delete items: 
    array.splice( start_index, num_of_items);
  • Add at the fornt: 
    array.unshift(new_first_item);
  • Delete the front: 
    array.shift();
  • Find an item: 
    array.indexOf(item_name);
  • copy part of the array: 
    array.slice( start_inclusive_index, end_exclusive_index);
  • Iterating an Array:
    • For Loop: 
      for( var i = 0; i < array.length; i++)
    • forEach: 
      array.forEach( function( array_item ){
        //do something
      });

JavaScript Basics: Obeject

Store data in key-value pairs.
  • Create an object: 
     var object_name{
       key_var_1: value_1, key_var_2: value2
    };
    or 
     var object_name = new Object();
  • Access properties: 
    object_name[property] or object_name.preperty
  • Array of Objects: 
    var array = [ { name: Yuling, age: 20}, { animal: dog, hair: bold}];
  • Function of an object: 
    var obj = { displayName: function(){ console.log("Yuling")}};
    object.displayName(); // Like instance method in Java
  • Keyword this: 
     obj_func = function(){ console.log( this.data); }
